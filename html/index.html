BIG
En desarrollo activo
📊
Big Data · Extensión Universitaria
SADESC

Sistema de Análisis de Datos para la Dirección de Extensión y Servicios a la Comunidad de UCLA. Procesamos grandes volúmenes de datos institucionales con pipelines Big Data para apoyar la toma de decisiones de extensión.

Capacidades del sistema
Ingesta de datos: Pipelines con Apache Hadoop y Spark para fuentes institucionales heterogéneas
Procesamiento distribuido: Transformación y limpieza de datos a escala con PySpark
Análisis estadístico: Modelos exploratorios y descriptivos sobre indicadores de extensión
Visualización: Dashboards interactivos con métricas de impacto para directivos
Equipos: HADOOPERS + CLOUDY · Código ISE009-2025-EXT-001 · 10 semanas
Python Apache Hadoop Apache Spark PySpark Big Data Scrum

CRM
Discovery completada
🏛️
Proyecto de Extensión Universitaria
CRM Extensión UCLA

Centralizamos en una sola plataforma web toda la operación de la Dirección de Extensión: solicitudes, eventos, finanzas, alianzas y certificados. Cero Excel, cero silos de información.

Módulos del sistema
CRM Comercial: Cuentas, contactos, oportunidades y pipeline institucional
Gestion de Eventos: Registro, matriculas por segmento, check-in y seguimiento
Motor Financiero: Presupuestos, tarifas diferenciadas, excedentes por sede
Alianzas y Convocatorias: Directorio de aliados, vigencias y alertas automáticas
Certificados Digitales: Generación automática PDF + insignias con ACREDITTA
Dashboard BI: KPIs en tiempo real filtrables por sede, facultad y evento
Java Spring Boot Angular MySQL REST API ACREDITTA

RAP
En desarrollo activo
📊
Software Academico y Analitica
Plan Assessment 2025

Automatizamos el cruce de calificaciones con rúbricas RAP y generamos reportes listos para acreditación en segundos. Del Excel manual al dashboard interactivo, sin fricción.

Capacidades del sistema
Cruce automatico RAP: Mapeo inteligente entre calificaciones detalladas y rubricas
Dashboard analitico: Visualizacion interactiva por grupo, cohorte y criterio
Exportación inteligente: Reportes listos para comités de acreditación y autoevaluación
Arquitectura ligera: Procesamiento local sin instalacion compleja, salida portable
6 Sprints Scrum: MVP funcional desde Sprint 3 con validación docente real
Python Pandas Power BI Star Schema Scrum ISO 27001

Proyectos de investigación: Semilleros, grupos y convocatorias internas gestionadas
Panel institucional: Indicadores para vicerrectoria, facultades y programas

Software de Gestión de Espacios
Reserva Dinámica de Espacios

Plataforma web que permite a estudiantes, docentes y administrativos visualizar la disponibilidad de espacios en tiempo real, hacer reservas dinámicas y optimizar el uso de la infraestructura física de la universidad.

Funcionalidades clave
Reserva en tiempo real: Visualización inmediata de disponibilidad
Panel administrativo: Control total sobre espacios y horarios
Notificaciones automáticas: Confirmación y recordatorios al usuario
Historial de uso: Métricas de ocupación por espacio y período
Vue.js Node.js MongoDB Vercel

INT
MVP disponible

Software de Gestión Académica
Internacionalización

Sistema digital que centraliza la gestión de convenios internacionales, movilidad estudiantil y docente, doble titulación y acreditaciones internacionales, facilitando los procesos de internacionalización de la Universidad Católica Luis Amigó.

Módulos del sistema
Gestión de convenios: Registro y seguimiento de alianzas internacionales
Movilidad estudiantil: Control de procesos de intercambio y becas
Doble titulación: Gestión de programas conjuntos con universidades extranjeras
Dashboard analítico: Indicadores de internacionalización por facultad
React Django PostgreSQL Vercel
